## Internal Linking Best Practices

### Priority Levels

**Tier 1 — Link Often** (every relevant article):
- Map customiser / create page
- Relevant gift occasion pages

**Tier 2 — Link Frequently** (most articles):
- Relevant map style pages
- Related blog articles in the same topic cluster

**Tier 3 — Link Occasionally**:
- About / brand story page (when heritage is relevant)
- Sizing guide (when dimensions are discussed)
- Help / FAQ (when ordering or delivery is mentioned)

**Tier 4 — Link Rarely**:
- Homepage
- Contact page

### Linking Guidelines

**Do**:
- Link when it genuinely adds value for the reader
- Use descriptive, specific anchor text
- Vary anchor text for the same destination
- Link to the customiser with invitation language ("design your map" not "click here")
- Include at least one link in the first 400 words of any article

**Don't**:
- Force links where they feel awkward
- Use generic anchor text ("click here", "this page", "read more")
- Add more than 2 links per paragraph
- Link to the same page more than once per article
- Use the exact same anchor text every time you link to the same page
