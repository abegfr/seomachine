# Target Keywords & Topic Clusters — CoolMaps

This document organises CoolMaps's target keywords by topic cluster for strategic content planning and SEO optimisation.

**Note**: Volume and difficulty estimates are indicative — verify with Ahrefs, SEMrush, or Google Keyword Planner before committing to content. Update regularly as you build ranking history.

---

## Topic Cluster Structure

Each cluster has:
- **Pillar Keyword**: Main, higher-volume keyword (typically more competitive)
- **Cluster Keywords**: 5–10 related keywords (subtopics)
- **Long-Tail Keywords**: 10–15 specific, lower-volume phrases (easier to rank for early)
- **Search Intent**: What users want (informational, commercial, transactional)

## Usage Guidelines

### When Writing New Content
1. Check which cluster the topic belongs to
2. Target the appropriate cluster or long-tail keyword
3. Reference the pillar content in that cluster with an internal link
4. Include LSI keywords naturally throughout
5. Link to related cluster articles

### When Planning the Content Calendar
1. Start with High Priority keywords — build early wins
2. Systematically build out gift occasion cluster (anniversary → wedding → housewarming → birthday)
3. Build location-specific content in parallel — these convert well with lower effort
4. Portugal heritage content for brand depth — even 1–2 articles here differentiates strongly
